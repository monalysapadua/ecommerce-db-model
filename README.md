⭐ Descrição do Projeto (para o GitHub)

Projeto de modelagem e implementação do banco de dados lógico para um sistema de e-commerce.
Inclui o mapeamento EER, criação do esquema relacional com PKs, FKs, constraints e refinamentos (Cliente PF/PJ, múltiplas formas de pagamento, status de entrega, rastreamento, fornecedores, vendedores, estoque e pedidos).
Também contém scripts SQL completos para criação das tabelas e queries avançadas para análise de dados.
